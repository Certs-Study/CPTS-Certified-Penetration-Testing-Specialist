# Burp Shortcuts

## Burp Shortcuts

| **Shortcut**      | **Description**  |
| ----------------- | ---------------- |
| \[`CTRL+R`]       | Send to repeater |
| \[`CTRL+SHIFT+R`] | Go to repeater   |
| \[`CTRL+I`]       | Send to intruder |
| \[`CTRL+SHIFT+I`] | Go to intruder   |
| \[`CTRL+U`]       | URL encode       |
| \[`CTRL+SHIFT+U`] | URL decode       |
